# V1 - 기본 시맨틱 검색 시스템

초기 구현된 기본 파이프라인 버전입니다.

## 특징
- 기본적인 시맨틱 검색 기능
- 단순한 텍스트 전처리
- 분리된 모듈 구조

## 파일 구성
- `main.py` - 메인 파이프라인 (데모용)
- `embedding.py` - 기본 임베딩 생성
- `preprocessing.py` - 기본 텍스트 전처리
- `segment_extraction.py` - 최적 구간 추출
- `similarity.py` - 코사인 유사도 계산

## 사용법

### 1. 자막 다운로드
```bash
python fetch_subtitle.py
```

### 2. 기본 파이프라인 실행
```bash
python main.py
```

## 특징
- **장점**: 단순하고 이해하기 쉬운 구조
- **단점**: 키워드 정확도 낮음, 제품 특화 기능 없음

## 한계점
- 직접 키워드 매칭 부족
- 제품 관련 키워드 사전 부재
